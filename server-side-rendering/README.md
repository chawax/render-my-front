# Server Side Rendering

Exemple d'application avec rendu côté serveur (SSR).

Stack technique :

- Next.js
- Typescript
- Tailwind

Installation :

```bash
npm install
```

Exécution en mode développement :

```bash
npm run dev
```

Puis accès à `http://localhost:3000`

Exécution en mode production :

```bash
npm run build
npm run start
```

Puis accès à `http://localhost:4300`
